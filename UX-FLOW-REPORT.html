Primary User Journeys

Journey 1: Guest Shopping (Most Common) 
index.html → Browse Categories → category.html
           → Click Product → product-detail.html
           → Add to Cart → cart.html
           → Checkout → checkout-guest.html
           → Payment → payment.html
           → Success → order-confirmation.html
Journey 2: Registered User Shopping 
login.html → index.html (logged in)
          → Browse → category.html
          → Product → product-detail-logged-in.html
          → Add to Wishlist (instant save)
          → Cart → cart.html
          → Checkout → checkout.html (saved addresses)
          → Payment → payment.html
          → Success → order-confirmation.html
          → Track → track-order.html
Journey 3: Account Management 
my-profile.html → Profile Management
                → my-orders.html (Order History)
                → order-detail.html (Specific Order)
                → track-order.html (Tracking)
                → my-reviews.html (Write Reviews)
                → wishlist.html (Saved Items)
                → settings.html (Preferences)
